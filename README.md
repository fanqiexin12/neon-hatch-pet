# Neon Hatch | 赛博像素宠物舱

一个静态网页电子宠物原型：宠物初始为蛋，可以设置孵化时间；孵化后随机生成宠物，并支持喂食、玩耍和心情互动反馈。

## 运行

```bash
python3 -m http.server 5173
```

然后打开：

```text
http://127.0.0.1:5173
```

## 功能

- 自定义孵化时间，倒计时完成后随机孵化宠物
- 随机宠物身份、性格和像素外观
- 宠物昵称、等级、经验和信用点会保存到本地
- 饱腹、快乐、能量、羁绊、清洁五项状态
- 喂食、玩耍、清洁、休息和训练会改变状态并记录日志
- 追光小游戏：限时点击画布里的发光核心，按命中奖励经验和信用点
- 外出探险：随机触发城市事件，记录发现地点并可能解锁装扮
- 补给商店可以购买强化能量饼、激光玩具和共情芯片
- 装扮系统：可购买或解锁星环、光冠、量子围巾并显示在像素宠物身上
- 每日任务会追踪喂食、玩耍、心情互动、训练、追光和探险，完成后可领取奖励
- 输入心情后，宠物会根据关键词给出反馈
- 暴露 `window.render_game_to_text()` 和 `window.advanceTime(ms)` 便于浏览器自动化验证
