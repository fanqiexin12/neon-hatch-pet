# Neon Hatch | 赛博像素宠物舱

一个赛博像素电子宠物项目。v1 是网页宠物舱；v2 新增真正的桌面宠物窗口，宠物可以透明置顶显示在桌面上。

## 运行

### 桌面宠物版

首次运行先安装依赖：

```bash
npm install
```

启动桌面宠物：

```bash
npm run desktop
```

桌面版特性：

- 透明、无边框、置顶的小窗口
- 顶部可拖动，右键可打开菜单、回到右下角或退出
- 初始为蛋，点击孵化后随机出现宠物；已孵化时可点“重孵”回到蛋
- 支持喂食、玩耍、探险、摸摸、跳舞、睡觉、清洁和心情选项回应
- 点击像素宠物本体可以直接摸摸它
- 不用输入文字，可直接选择开心、伤心、焦虑、累了、生气、无聊、孤单、兴奋、平静、压力
- 孵化后会在空闲时自动随机做动作：环顾、小跳、挥手、伸展、摇摆
- 不同活动会触发不同像素动画：食物块、激光点、爱心、音符、睡觉 Z、清洁泡泡和安慰贴近动作
- 本地保存桌面宠物状态

### 网页宠物舱

```bash
python3 -m http.server 5173
```

然后打开：

```text
http://127.0.0.1:5173
```

## 功能

- 桌面版：透明置顶宠物窗口，可拖动、右键菜单、状态保存、重孵回蛋、心情选项回应和自动随机动作
- 自定义孵化时间，倒计时完成后随机孵化宠物
- 随机宠物身份、性格和像素外观
- 宠物昵称、等级、经验和信用点会保存到本地
- 饱腹、快乐、能量、羁绊、清洁五项状态
- 喂食、玩耍、清洁、休息和训练会改变状态并记录日志
- 追光小游戏：限时点击画布里的发光核心，按命中奖励经验和信用点
- 外出探险：随机触发城市事件，记录发现地点并可能解锁装扮
- 补给商店可以购买强化能量饼、激光玩具和共情芯片
- 装扮系统：可购买或解锁星环、光冠、量子围巾并显示在像素宠物身上
- 每日任务会追踪喂食、玩耍、心情互动、训练、追光和探险，完成后可领取奖励
- 输入心情后，宠物会根据关键词给出反馈
- 暴露 `window.render_game_to_text()` 和 `window.advanceTime(ms)` 便于浏览器自动化验证
